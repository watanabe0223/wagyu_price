# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
* リモートリポジトリ
ネット上に配置し複数人で作業を共有するためのリポジトリ
* ローカルリポジトリ
自分のPC状に保存し一人で作業するためのリポジトリ

## プッシュとマージの違いは何でしょうか？
* プッシュ
ローカルリポジトリからリモートリポジトリにアップロードする事
* マージ
派生したブランチを統合すること


## コミットとプッシュの違い
* コミット
変更を行ったリポジトリの保存を行う事
* プッシュ
コミットを行ったリポジトリをプッシュする



## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
* 変更を行ったなどのメッセージではなく、誰がみてもわかるようなメッセージを記述する



## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
* ローカルでマージするのは作業者が直接マスターにマージするのに対し<br>プルリクエストでは、ブランチをリモートにプッシュしてから  レビュー担当者がマスターにマージする


## コンフリクトを起こしてしまった場合、どう対処すべきですか？
* 競合している箇所を修正する。複数人で作業を行っている場合は相談し修正する
